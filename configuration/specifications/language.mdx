All Print.App platforms uses AWS translation service to deliver highly accurate neural machine language translation that can be customized.

Print.App supports the following languages with their curresponding [ISO 639-1](https://www.loc.gov/standards/iso639-2/php/code_list.php) two-digit language codes.

For a country variant of a language, the table follows the [RFC 5646](https://www.rfc-editor.org/rfc/rfc5646.html) format of appending a dash followed by an [ISO 3166](https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes) 2-digit country code.

<Tip>
In our supported shopping cart frameworks, the plugin automatically gets the language code from the html page and pass it unto the editor
So you don't have to do anything on your part to enable language translations. Simply set your store to the language of choice
and the plugin will set the editor to that language.

</Tip>
