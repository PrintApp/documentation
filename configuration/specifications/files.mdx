### Downloads

| Extension   | -    | Mimetype          | Description        |
| --------    | ----| ----              |------------        |
| `pdf`       | -   |  application/pdf         | Rich, full vector PDF file |
| `png`       | -   |  image/png         | PNG Image with transparency |
| `jpeg`     | -   | image/jpeg        | Jpeg Image file |
| `tiff`      | -   | image/tiff         | TIFF file |
