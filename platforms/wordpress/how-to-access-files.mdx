---
description: "How to access completed order files in Wordpress"
---

In Wordpress, the project files can be accessed in your Wordpress Admin and are listed against each order.

Navigate to `WooCommerce` >> `Orders` and click on the order.

You will find the links to download the Print-Ready files against each order as shown below
